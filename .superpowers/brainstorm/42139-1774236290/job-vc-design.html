ConvertJobViewController UI 设计

一个 VC 管理四个状态，push 进入，背景与其他页面统一（MeshGradient）

状态机

idle
等待选文件
→
fileSelected
已选，待确认
→
converting
上传+处理中
→
success
转换完成
↘
error
失败可重试
← converting 失败时

各状态 UI 内容

idle 状态
  • 大图标 + 工具名称（与卡片一致）
  • 副标题说明支持的文件格式
  • 「选择文件」主按钮（渐变色）
  • 合并PDF：按钮文字改为「选择多个文件」
  • 导航栏有「取消」返回
fileSelected 状态
  • 文件卡片：图标 + 文件名 + 大小
  • 合并PDF：显示文件列表（可删除单个）
  • 格式标签（如 → PDF）显示在文件卡右侧
  • 「开始转换」主按钮
  • 「重新选择」次要按钮
converting 状态
  • 两阶段进度：
    ① 上传进度条（URLSession 精确进度）
    ② 服务端处理中（不确定进度，动画条）
  • 文字提示："正在上传..." / "转换中..."
  • 「取消」按钮（取消任务）
  • 禁用返回手势
success 状态
  • 绿色勾选动画
  • 「预览结果」主按钮 → present QLPreviewController
  • 「再转一个」次要按钮 → 重置回 idle
  • QLPreviewController 提供系统分享/存储
error 状态
  • 红色感叹号图标
  • 显示错误信息（来自 API msg 字段）
  • 「重试」主按钮 → 回到 fileSelected
  • 「重新选择文件」次要按钮 → idle

完整交互流程

有格式选项的工具：
点击卡片 → UIAlertController(actionSheet) 选格式 → UIDocumentPickerViewController 选文件 → push ConvertJobViewController(fileSelected) → 点「开始转换」→ converting → success → QLPreviewController

无格式选项的工具（PDF转Word / 文件转图片）：
点击卡片 → UIDocumentPickerViewController 选文件 → push ConvertJobViewController(fileSelected) → 点「开始转换」→ converting → success → QLPreviewController

合并 PDF：
点击卡片 → UIDocumentPickerViewController(多选) 选文件 → push ConvertJobViewController(fileSelected，显示文件列表) → 点「开始转换」→ 自动判断调 mergetwopdf 或 mergemorepdf → success → QLPreviewController
