新增文件

新文件 1

ConvertAPIService.swift

格式转换网络层，multipart/form-data 上传 + 下载结果文件

  • 支持 PDF / Word / Excel / PPT 转换
  • 支持合并 PDF（2 个或多个）
  • 支持文件转图片
  • URLSession 上传进度回调
  • 解析 newslist → 下载 URL
新文件 2

ConvertJobViewController.swift

单次转换任务 VC，管理完整流程状态机

  • State: idle → fileSelected → converting → success / error
  • 显示文件名、大小、格式标签
  • 进度条（上传进度 + 处理中动画）
  • 转换成功后 present QLPreviewController
  • 错误时显示原因 + 重试按钮
