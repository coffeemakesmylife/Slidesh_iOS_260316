转换功能：输出格式如何选择？

对于支持多种输出格式的工具（如"PDF 转换器"支持 Word/Excel/PPT/HTML/PNG），用户应在哪个步骤选择格式？

A

点击工具卡片后立即选择格式

点击工具 → 底部弹窗选择输出格式 → 选文件 → 上传转换

优点

  • 用户先明确目标再找文件，逻辑清晰
  • 减少返回操作

缺点

  • 多一步才能到文件选择器
B

选完文件后再选择格式

点击工具 → 选文件 → 底部弹窗选择输出格式 → 上传转换

优点

  • 直接进入文件选择，少一次点击

缺点

  • 用户不知道要转成什么格式就去找文件，稍显反逻辑
C

在工具卡片上直接展示格式子按钮

工具卡展开后显示格式按钮（PDF / Word / HTML …）→ 点格式 → 选文件 → 转换

优点

  • 一屏可见所有格式，最直观

缺点

  • 格式多时卡片会很高，影响列表视觉
  • 改变了原有卡片风格
